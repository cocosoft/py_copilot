# 知识管理向量化管理模块优化建议

## 1. 概述

### 1.1 背景

当前项目的知识管理系统采用分层架构，包含文档级、知识库级和全局级三层知识图谱体系。向量化管理作为知识管理的核心组件，负责将文档内容转换为向量表示，支持语义搜索和相似度匹配。

### 1.2 现有架构分析

当前向量化管理模块主要包含以下核心组件：

| 组件 | 功能 | 技术实现 |
|------|------|----------|
| DocumentProcessor | 文档处理核心 | 文档解析、分块、向量化 |
| ChromaService | 向量存储服务 | 基于ChromaDB的HTTP API客户端 |
| FAISSIndexService | 高性能索引 | FAISS向量索引 |
| RetrievalService | 检索服务 | 基础向量检索 |
| SemanticSearchService | 语义搜索 | 增强语义理解和搜索 |

## 2. 当前存在的问题

### 2.1 性能瓶颈

#### 2.1.1 批量处理效率

当前`document_processor.py`中的向量化处理采用分批策略：

```python
# 当前实现：固定批次大小
if total_chunks <= 50:
    batch_size = total_chunks
elif total_chunks <= 100:
    batch_size = 50
else:
    batch_size = 50
```

**问题**：
- 批次大小固定，未根据系统负载动态调整
- 大批量文档处理时内存占用过高
- 缺乏流式处理机制

#### 2.1.2 向量检索延迟

- 高维向量相似度计算耗时较长
- 缺乏索引预热机制
- 未实现查询结果缓存

### 2.2 数据一致性问题

#### 2.2.1 向量与元数据同步

当前设计中，向量存储在ChromaDB，元数据存储在关系型数据库，存在双写一致性问题：

```python
# 潜在问题场景
# 1. 向量写入成功，但数据库事务回滚
# 2. 文档删除时，向量可能残留
```

#### 2.2.2 版本管理缺失

- 文档更新时，历史向量未清理
- 缺乏向量版本追踪机制

### 2.3 扩展性限制

#### 2.3.1 单点服务依赖

```python
class ChromaService:
    def __init__(self, server_url: str = "http://localhost:8008"):
        self.server_url = server_url
```

- 依赖单一ChromaDB服务实例
- 缺乏服务发现和负载均衡
- 不支持水平扩展

#### 2.3.2 模型耦合

- 嵌入模型配置硬编码
- 不支持多模型动态切换
- 缺乏模型版本管理

### 2.4 可观测性不足

- 缺乏向量操作详细指标监控
- 搜索质量无法量化评估
- 未实现向量分布可视化

## 3. 优化建议

### 3.1 架构优化

#### 3.1.1 引入向量数据库抽象层

建议设计统一的向量存储接口，支持多种后端：

```
└─────────────────────────────────────────────────────────────┘
                              │
            ┌─────────────────┼─────────────────┐

   │   写入服务    │  │   查询服务    │  │   管理服务    │

          │                 │                 │
          ▼                 ▼                 ▼
   ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
   │  主向量库     │  │  只读副本     │  │  元数据存储   │
   │  (Primary)   │  │  (Replicas)  │  │ (Metadata)   │
   └──────────────┘  └──────────────┘  └──────────────┘
```

**优化点**：
- 写入操作走主库，保证一致性
- 查询操作分发到只读副本，提升并发
- 管理操作独立，避免影响业务

### 3.2 性能优化

#### 3.2.1 动态批次处理

优化`DocumentProcessor`的分批策略：

```python
import psutil
import asyncio
from dataclasses import dataclass
from typing import Callable


@dataclass
class BatchConfig:
    """动态批次配置"""
    min_batch_size: int = 10
    max_batch_size: int = 100
    memory_threshold: float = 0.8  # 内存使用阈值
    processing_timeout: float = 30.0


class AdaptiveBatchProcessor:
    """自适应批次处理器"""

    def __init__(self, config: BatchConfig = None):
        self.config = config or BatchConfig()
        self._metrics = {
            'processed_count': 0,
            'failed_count': 0,
            'avg_processing_time': 0.0
        }

    def calculate_optimal_batch_size(self) -> int:
        """根据系统负载计算最优批次大小"""
        memory_usage = psutil.virtual_memory().percent / 100

        if memory_usage > self.config.memory_threshold:
            # 内存压力大，减小批次
            return max(
                self.config.min_batch_size,
                int(self.config.max_batch_size * 0.5)
            )

        # 根据历史处理时间调整
        if self._metrics['avg_processing_time'] > self.config.processing_timeout:
            return max(
                self.config.min_batch_size,
                int(self._metrics['processed_count'] / 2)
            )

        return self.config.max_batch_size

    async def process_batches(
        self,
        items: List[Any],
        processor: Callable[[List[Any]], Any],
        progress_callback: Callable[[int, int], None] = None
    ) -> Dict[str, Any]:
        """自适应批次处理"""
        results = []
        failed_items = []
        total = len(items)
        processed = 0

        while processed < total:
            batch_size = self.calculate_optimal_batch_size()
            batch = items[processed:processed + batch_size]

            try:
                start_time = asyncio.get_event_loop().time()
                batch_result = await processor(batch)
                elapsed = asyncio.get_event_loop().time() - start_time

                # 更新指标
                self._update_metrics(len(batch), elapsed)
                results.extend(batch_result)

            except Exception as e:
                # 批次失败，降级为逐个处理
                logger.warning(f"批次处理失败，降级处理: {e}")
                for item in batch:
                    try:
                        result = await processor([item])
                        results.extend(result)
                    except Exception as item_error:
                        failed_items.append({
                            'item': item,
                            'error': str(item_error)
                        })

            processed += len(batch)
            if progress_callback:
                progress_callback(processed, total)

        return {
            'success_count': len(results),
            'failed_count': len(failed_items),
            'results': results,
            'failed_items': failed_items
        }

    def _update_metrics(self, count: int, elapsed: float):
        """更新处理指标"""
        self._metrics['processed_count'] += count

        # 指数移动平均
        alpha = 0.3
        current_avg = self._metrics['avg_processing_time']
        self._metrics['avg_processing_time'] = (
            alpha * (elapsed / count) + (1 - alpha) * current_avg
        )
```

#### 3.2.3 查询缓存机制

```python
import hashlib
import json
from functools import wraps
from datetime import datetime, timedelta
from typing import Dict, Any, Optional
import asyncio


class VectorQueryCache:
    """向量查询缓存"""

    def __init__(self, ttl: int = 300, max_size: int = 10000):
        self.ttl = ttl
        self.max_size = max_size
        self._cache: Dict[str, Dict] = {}
        self._access_count: Dict[str, int] = {}
        self._lock = asyncio.Lock()

    def _generate_key(
        self,
        query_vector: Any,
        top_k: int,
        filters: Optional[Dict]
    ) -> str:
        """生成缓存键"""
        cache_data = {
            'vector_hash': hashlib.md5(
                query_vector.tobytes()
            ).hexdigest()[:16],
            'top_k': top_k,
            'filters': filters
        }
        return hashlib.md5(
            json.dumps(cache_data, sort_keys=True).encode()
        ).hexdigest()

    async def get(
        self,
        query_vector: Any,
        top_k: int,
        filters: Optional[Dict] = None
    ) -> Optional[List[Dict]]:
        """获取缓存结果"""
        key = self._generate_key(query_vector, top_k, filters)

        async with self._lock:
            if key in self._cache:
                entry = self._cache[key]
                if datetime.now() < entry['expires_at']:
                    self._access_count[key] += 1
                    return entry['results']
                else:
                    # 过期清理
                    del self._cache[key]
                    del self._access_count[key]

        return None

    async def set(
        self,
        query_vector: Any,
        top_k: int,
        filters: Optional[Dict],
        results: List[Dict]
    ):
        """设置缓存"""
        key = self._generate_key(query_vector, top_k, filters)

        async with self._lock:
            # 容量控制
            if len(self._cache) >= self.max_size:
                # LRU淘汰
                lru_key = min(
                    self._access_count.keys(),
                    key=lambda k: self._access_count[k]
                )
                del self._cache[lru_key]
                del self._access_count[lru_key]

            self._cache[key] = {
                'results': results,
                'expires_at': datetime.now() + timedelta(seconds=self.ttl),
                'created_at': datetime.now()
            }
            self._access_count[key] = 1


def cached_vector_search(ttl: int = 300):
    """向量搜索缓存装饰器"""
    cache = VectorQueryCache(ttl=ttl)

    def decorator(func):
        @wraps(func)
        async def wrapper(
            self,
            query: str,
            top_k: int = 10,
            **kwargs
        ):
            # 获取查询向量
            query_vector = await self._get_query_embedding(query)

            # 尝试从缓存获取
            cached = await cache.get(query_vector, top_k, kwargs.get('filters'))
            if cached is not None:
                return cached

            # 执行搜索
            results = await func(self, query, top_k, **kwargs)

            # 缓存结果
            await cache.set(query_vector, top_k, kwargs.get('filters'), results)

            return results
        return wrapper
    return decorator
```

### 3.3 数据一致性优化

#### 3.3.1 事务性向量操作

```python
from contextlib import asynccontextmanager
from typing import AsyncGenerator
import asyncio


class VectorTransaction:
    """向量操作事务管理器"""

    def __init__(self, vector_store, db_session):
        self.vector_store = vector_store
        self.db_session = db_session
        self._operations = []
        self._committed = False
        self._lock = asyncio.Lock()

    async def add_document(
        self,
        doc_id: str,
        vector: np.ndarray,
        metadata: Dict[str, Any]
    ):
        """添加文档（记录操作）"""
        async with self._lock:
            self._operations.append({
                'type': 'add',
                'doc_id': doc_id,
                'vector': vector,
                'metadata': metadata
            })

    async def delete_document(self, doc_id: str):
        """删除文档（记录操作）"""
        async with self._lock:
            self._operations.append({
                'type': 'delete',
                'doc_id': doc_id
            })

    async def commit(self):
        """提交事务"""
        async with self._lock:
            if self._committed:
                raise RuntimeError("事务已提交")

            try:
                # 1. 先写入向量数据库
                for op in self._operations:
                    if op['type'] == 'add':
                        await self.vector_store.add_vectors(
                            [op['doc_id']],
                            op['vector'].reshape(1, -1),
                            [op['metadata']]
                        )
                    elif op['type'] == 'delete':
                        await self.vector_store.delete_vectors([op['doc_id']])

                # 2. 提交数据库事务
                await self.db_session.commit()
                self._committed = True

            except Exception as e:
                # 回滚操作
                await self.rollback()
                raise VectorTransactionError(f"事务提交失败: {e}")

    async def rollback(self):
        """回滚事务"""
        # 回滚数据库事务
        await self.db_session.rollback()

        # 补偿向量操作（如果可能）
        # 注意：向量数据库通常不支持真正的回滚
        # 这里可以记录补偿日志，由后台任务处理
        for op in reversed(self._operations):
            if op['type'] == 'add':
                # 记录需要清理的向量
                await self._record_compensation('delete', op['doc_id'])


@asynccontextmanager
async def vector_transaction(
    vector_store,
    db_session
) -> AsyncGenerator[VectorTransaction, None]:
    """事务上下文管理器"""
    tx = VectorTransaction(vector_store, db_session)
    try:
        yield tx
        await tx.commit()
    except Exception:
        await tx.rollback()
        raise
```

```python
from datetime import datetime
from typing import List, Optional
from dataclasses import dataclass, asdict
import json


@dataclass
class VectorVersion:
    """向量版本信息"""
    version_id: str
    document_id: str
    vector_id: str
    created_at: datetime
    created_by: str
    metadata_hash: str
    is_active: bool = True
    parent_version: Optional[str] = None


class VectorVersionManager:
    """向量版本管理器"""

    def __init__(self, db_session, vector_store):
        self.db_session = db_session
        self.vector_store = vector_store

    async def create_version(
        self,
        document_id: str,
        vector: np.ndarray,
        metadata: Dict[str, Any],
        created_by: str = "system"
    ) -> VectorVersion:
        """创建新版本"""

        # 生成版本ID
        version_id = self._generate_version_id(document_id)
        vector_id = f"{document_id}_{version_id}"

        # 计算元数据哈希
        metadata_hash = self._hash_metadata(metadata)

        # 检查是否有变化
        last_version = await self.get_latest_version(document_id)
        if last_version and last_version.metadata_hash == metadata_hash:
            return last_version  # 无变化，返回现有版本

        # 停用旧版本
        if last_version:
            await self._deactivate_version(last_version.version_id)

        # 创建新版本记录
        version = VectorVersion(
            version_id=version_id,
            document_id=document_id,
            vector_id=vector_id,
            created_at=datetime.now(),
            created_by=created_by,
            metadata_hash=metadata_hash,
            is_active=True,
            parent_version=last_version.version_id if last_version else None
        )

        # 保存到数据库
        await self._save_version_record(version)

        # 添加向量到存储
        await self.vector_store.add_vectors(
            [vector_id],
            vector.reshape(1, -1),
            [{**metadata, 'version_id': version_id}]
        )

        return version

    async def get_version_history(
        self,
        document_id: str
    ) -> List[VectorVersion]:
        """获取版本历史"""
        # 查询数据库获取版本历史
        pass

    async def rollback_to_version(
        self,
        document_id: str,
        version_id: str
    ) -> bool:
        """回滚到指定版本"""
        # 1. 获取目标版本
        target = await self._get_version(version_id)
        if not target:
            raise ValueError(f"版本不存在: {version_id}")

        # 2. 停用当前版本
        current = await self.get_latest_version(document_id)
        if current:
            await self._deactivate_version(current.version_id)

        # 3. 重新激活目标版本
        await self._activate_version(version_id)

        return True

    async def cleanup_old_versions(
        self,
        document_id: str,
        keep_count: int = 5
    ):
        """清理旧版本，只保留最近N个"""
        versions = await self.get_version_history(document_id)

        if len(versions) <= keep_count:
            return

        to_delete = versions[keep_count:]
        for version in to_delete:
            # 删除向量
            await self.vector_store.delete_vectors([version.vector_id])
            # 删除版本记录
            await self._delete_version_record(version.version_id)

    def _generate_version_id(self, document_id: str) -> str:
        """生成版本ID"""
        timestamp = datetime.now().strftime("%Y%m%d%H%M%S")
        return f"{document_id}_{timestamp}"

    def _hash_metadata(self, metadata: Dict) -> str:
        """计算元数据哈希"""
        return hashlib.md5(
            json.dumps(metadata, sort_keys=True).encode()
        ).hexdigest()
```

#### 3.4.1 向量操作指标收集

```python
from prometheus_client import Counter, Histogram, Gauge

class VectorMetricsCollector:
    """向量指标收集器"""

    @staticmethod
    def record_operation(operation: str):
        """操作记录装饰器"""
        def decorator(func):
            @wraps(func)
            async def wrapper(*args, **kwargs):
                start_time = time.time()
                status = 'success'

                try:
                    result = await func(*args, **kwargs)
                    return result
                except Exception as e:
                    status = 'error'
                    raise
                finally:
                    VECTOR_OPERATIONS.labels(
                        operation=operation,
                        status=status

                            top_k=str(kwargs.get('top_k', 10))
                        ).observe(time.time() - start_time)

            return wrapper
        return decorator

    @staticmethod

        """更新索引大小"""
        VECTOR_INDEX_SIZE.labels(collection=collection).set(size)

            'diversity_score': self._calculate_diversity(results),
            'relevance_spread': self._calculate_relevance_spread(results),
            'coverage_score': self._calculate_coverage(query, results)
        }

        if user_feedback:
            metrics['user_satisfaction'] = self._calculate_satisfaction(
                user_feedback
            )

        return metrics

    def _calculate_diversity(self, results: List[Dict]) -> float:
        """计算结果多样性"""
        if len(results) < 2:
            return 1.0

        # 基于向量相似度计算多样性
        vectors = [r.get('vector') for r in results if r.get('vector')]
        if len(vectors) < 2:
            return 1.0

        # 计算平均余弦相似度
        similarities = []
        for i in range(len(vectors)):
            for j in range(i + 1, len(vectors)):
                sim = self._cosine_similarity(vectors[i], vectors[j])
                similarities.append(sim)

        avg_sim = sum(similarities) / len(similarities)
        # 多样性 = 1 - 平均相似度
        return 1.0 - avg_sim

    def _calculate_relevance_spread(self, results: List[Dict]) -> float:
        """计算相关性分数分布"""
        if not results:
            return 0.0

        scores = [r.get('score', 0) for r in results]
        if len(scores) < 2:
            return 0.0

        # 计算分数标准差
        mean = sum(scores) / len(scores)
        variance = sum((s - mean) ** 2 for s in scores) / len(scores)
        std = variance ** 0.5

        # 归一化到0-1
        return min(std * 2, 1.0)

    def _calculate_coverage(
        self,
        query: str,
        results: List[Dict]
    ) -> float:
        """计算查询覆盖度"""
        query_terms = set(query.lower().split())

        covered_terms = set()
        for result in results:
            content = result.get('content', '').lower()
            for term in query_terms:
                if term in content:
                    covered_terms.add(term)

        return len(covered_terms) / len(query_terms) if query_terms else 0.0

    def _cosine_similarity(self, v1: np.ndarray, v2: np.ndarray) -> float:
        """计算余弦相似度"""
        dot = np.dot(v1, v2)
        norm1 = np.linalg.norm(v1)
        norm2 = np.linalg.norm(v2)
        return dot / (norm1 * norm2) if norm1 > 0 and norm2 > 0 else 0.0
```

#### 3.5.1 模型动态切换

```python
from typing import Dict, Any, Optional
from dataclasses import dataclass
import asyncio


@dataclass
class EmbeddingModelConfig:
    """嵌入模型配置"""
    model_name: str
    model_path: str
    dimension: int
    max_seq_length: int
    batch_size: int
    device: str = "cpu"
    priority: int = 0  # 优先级，数值越大优先级越高


class EmbeddingModelManager:
    """嵌入模型管理器"""

    def __init__(self):
        self._models: Dict[str, Any] = {}
        self._configs: Dict[str, EmbeddingModelConfig] = {}
        self._current_model: Optional[str] = None
        self._lock = asyncio.Lock()

    async def register_model(self, config: EmbeddingModelConfig, model_instance: Any):
        """注册模型"""
        async with self._lock:
            self._configs[config.model_name] = config
            self._models[config.model_name] = model_instance

            # 如果是第一个模型或优先级更高，设为默认
            if (self._current_model is None or
                config.priority > self._configs[self._current_model].priority):
                self._current_model = config.model_name

    async def switch_model(self, model_name: str) -> bool:
        """切换当前使用的模型"""
        async with self._lock:
            if model_name not in self._models:
                raise ValueError(f"模型未注册: {model_name}")

            old_model = self._current_model
            self._current_model = model_name

            # 记录切换日志
            logger.info(f"模型切换: {old_model} -> {model_name}")

            return True

    async def get_embedding(
        self,
        text: str,
        model_name: Optional[str] = None
    ) -> np.ndarray:
        """获取文本嵌入向量"""

        target_model = model_name or self._current_model

        if target_model not in self._models:
            raise ValueError(f"模型未找到: {target_model}")

        model = self._models[target_model]
        config = self._configs[target_model]

        # 文本截断
        if len(text) > config.max_seq_length * 4:  # 粗略估计
            text = text[:config.max_seq_length * 4]

        # 获取嵌入
        embedding = await self._encode(model, text)

        return embedding

    async def get_embeddings_batch(
        self,
        texts: List[str],
        model_name: Optional[str] = None
    ) -> np.ndarray:
        """批量获取嵌入向量"""

        target_model = model_name or self._current_model
        config = self._configs[target_model]
        model = self._models[target_model]

        # 分批处理
        all_embeddings = []
        for i in range(0, len(texts), config.batch_size):
            batch = texts[i:i + config.batch_size]
            embeddings = await self._encode_batch(model, batch)
            all_embeddings.append(embeddings)

        return np.vstack(all_embeddings)

    async def _encode(self, model: Any, text: str) -> np.ndarray:
        """单文本编码（子类实现）"""
        raise NotImplementedError

    async def _encode_batch(self, model: Any, texts: List[str]) -> np.ndarray:
        """批量编码（子类实现）"""
        raise NotImplementedError

    def get_model_info(self) -> Dict[str, Any]:
        """获取模型信息"""
        return {
            'current_model': self._current_model,
            'available_models': [
                {
                    'name': name,
                    'dimension': config.dimension,
                    'device': config.device,
                    'priority': config.priority,
                    'is_active': name == self._current_model
                }
                for name, config in self._configs.items()
            ]
        }
```

```jsx
    mutationFn: vectorizeDocument,
    onMutate: async (docId) => {

      queryClient.setQueryData(['documents'], context.previousDocs);
```

```jsx
        nodes.push({
          id: `doc-${doc.id}`,
          name: doc.title,
          type: 'document',
          val: doc.chunk_count || 1
        });
      });

      documents.forEach((doc1, i) => {
        documents.slice(i + 1).forEach(doc2 => {
          const similarity = calculateDocSimilarity(doc1, doc2);
          if (similarity >= similarityThreshold) {
            links.push({
              source: `doc-${doc1.id}`,
              target: `doc-${doc2.id}`,
              value: similarity
            });
          }
        });
      });

};
```

```jsx
// components/Vectorization/ChunkEditor.jsx
import React, { useState, useCallback } from 'react';
import { DndProvider, useDrag, useDrop } from 'react-dnd';
import { HTML5Backend } from 'react-dnd-html5-backend';

/**
 * 交互式片段编辑器
 * 
 * 允许用户手动调整、合并、拆分向量化后的片段
 */
const ChunkEditor = ({ document, chunks, onSave }) => {
  const [editedChunks, setEditedChunks] = useState(chunks);
  const [selectedChunks, setSelectedChunks] = useState([]);
  const [splitPosition, setSplitPosition] = useState(null);
  const [history, setHistory] = useState([chunks]);
  const [historyIndex, setHistoryIndex] = useState(0);

  // 撤销操作
  const undo = () => {
    if (historyIndex > 0) {
      const newIndex = historyIndex - 1;
      setHistoryIndex(newIndex);
      setEditedChunks(history[newIndex]);
    }
  };

  // 重做操作
  const redo = () => {
    if (historyIndex < history.length - 1) {
      const newIndex = historyIndex + 1;
      setHistoryIndex(newIndex);
      setEditedChunks(history[newIndex]);
    }
  };

  // 保存状态到历史
  const saveToHistory = (newChunks) => {
    const newHistory = history.slice(0, historyIndex + 1);
    newHistory.push(newChunks);
    setHistory(newHistory);
    setHistoryIndex(newHistory.length - 1);
    setEditedChunks(newChunks);
  };

  // 合并选中的片段
  const mergeSelectedChunks = () => {
    if (selectedChunks.length < 2) return;

    const sortedIndices = selectedChunks.sort((a, b) => a - b);
    const firstIndex = sortedIndices[0];
    
    const mergedContent = sortedIndices
      .map(i => editedChunks[i].content)
      .join('\n\n');
    
    const mergedChunk = {
      ...editedChunks[firstIndex],
      content: mergedContent,
      char_count: mergedContent.length,
      merged_from: sortedIndices.map(i => editedChunks[i].id)
    };

    const newChunks = editedChunks.filter((_, i) => !selectedChunks.includes(i));
    newChunks.splice(firstIndex, 0, mergedChunk);
    
    saveToHistory(newChunks);
    setSelectedChunks([]);
  };

  // 拆分片段
  const splitChunk = (index, position) => {
    const chunk = editedChunks[index];
    const content = chunk.content;
    
    const part1 = content.substring(0, position);
    const part2 = content.substring(position);
    
    const newChunks = [...editedChunks];
    newChunks.splice(index, 1, 
      { ...chunk, id: `${chunk.id}_1`, content: part1, char_count: part1.length },
      { ...chunk, id: `${chunk.id}_2`, content: part2, char_count: part2.length, split_from: chunk.id }
    );
    
    saveToHistory(newChunks);
    setSplitPosition(null);
  };

  // 删除片段
  const deleteChunk = (index) => {
    const newChunks = editedChunks.filter((_, i) => i !== index);
    saveToHistory(newChunks);
  };

  // 重新排序
  const moveChunk = useCallback((dragIndex, hoverIndex) => {
    const draggedChunk = editedChunks[dragIndex];
    const newChunks = [...editedChunks];
    newChunks.splice(dragIndex, 1);
    newChunks.splice(hoverIndex, 0, draggedChunk);
    saveToHistory(newChunks);
  }, [editedChunks]);

  return (
    <DndProvider backend={HTML5Backend}>
      <div className="chunk-editor">
        {/* 工具栏 */}
        <div className="editor-toolbar">
          <div className="toolbar-group">
            <button 
              onClick={undo} 
              disabled={historyIndex === 0}
              title="撤销 (Ctrl+Z)"
            >
              ↩️ 撤销
            </button>
            <button 
              onClick={redo} 
              disabled={historyIndex === history.length - 1}
              title="重做 (Ctrl+Y)"
            >
              ↪️ 重做
            </button>
          </div>

          <div className="toolbar-group">
            <button 
              onClick={mergeSelectedChunks}
              disabled={selectedChunks.length < 2}
              title="合并选中的片段"
            >
              🔗 合并 ({selectedChunks.length})
            </button>
            <button 
              onClick={() => setSelectedChunks([])}
              disabled={selectedChunks.length === 0}
            >
              ❌ 清除选择
            </button>
          </div>

          <div className="toolbar-group">
            <button onClick={() => onSave(editedChunks)} className="save-btn">
              💾 保存修改
            </button>
          </div>
        </div>

        {/* 文档信息 */}
        <div className="document-info">
          <h4>{document.title}</h4>
          <span>原始片段数: {chunks.length}</span>
          <span>当前片段数: {editedChunks.length}</span>
          <span>已修改: {historyIndex > 0 ? '是' : '否'}</span>
        </div>

        {/* 片段列表 */}
        <div className="chunks-container">
          {editedChunks.map((chunk, index) => (
            <DraggableChunk
              key={chunk.id}
              index={index}
              chunk={chunk}
              isSelected={selectedChunks.includes(index)}
              onSelect={() => {
                setSelectedChunks(prev => 
                  prev.includes(index)
                    ? prev.filter(i => i !== index)
                    : [...prev, index]
                );
              }}
              onDelete={() => deleteChunk(index)}
              onSplit={(pos) => splitChunk(index, pos)}
              moveChunk={moveChunk}
            />
          ))}
        </div>

        {/* 操作提示 */}
        <div className="editor-hints">
          <p>💡 提示：</p>
          <ul>
            <li>点击片段可选中，按住 Ctrl 可多选</li>
            <li>拖拽片段可调整顺序</li>
            <li>在片段内点击可设置拆分位置</li>
            <li>修改后记得点击保存</li>
          </ul>
        </div>

/**
 * 可拖拽的片段组件
 */
const DraggableChunk = ({ 
  index, 
  chunk, 
  isSelected, 
  onSelect, 
  onDelete, 
  onSplit,
  moveChunk 
}) => {
  const [{ isDragging }, drag] = useDrag({
    type: 'chunk',
    item: { index },
    collect: (monitor) => ({
      isDragging: monitor.isDragging()
    })
  });

  const [, drop] = useDrop({
    accept: 'chunk',
    hover: (item) => {
      if (item.index !== index) {
        moveChunk(item.index, index);
        item.index = index;
      }
    }
  });

  const [splitPoint, setSplitPoint] = useState(null);

  return (
    <div
      ref={(node) => drag(drop(node))}
      className={`chunk-item ${isSelected ? 'selected' : ''} ${isDragging ? 'dragging' : ''}`}
      onClick={(e) => {
        if (e.ctrlKey || e.metaKey) {
          onSelect();
        }
      }}
    >
      <div className="chunk-header">
        <input 
          type="checkbox" 
          checked={isSelected}
          onChange={onSelect}
        />
        <span className="chunk-number">片段 {index + 1}</span>
        <span className="chunk-stats">
          {chunk.char_count} 字符 | {chunk.token_count || '?'} tokens
        </span>
        <button className="delete-btn" onClick={onDelete} title="删除">
          🗑️
        </button>
      </div>

      <div className="chunk-content-wrapper">
        <textarea
          className="chunk-content"
          value={chunk.content}
          readOnly
          onClick={(e) => {
            const pos = e.target.selectionStart;
            setSplitPoint(pos);
          }}
        />
        
        {splitPoint !== null && (
          <div className="split-actions">
            <span>在位置 {splitPoint} 拆分?</span>
            <button onClick={() => onSplit(splitPoint)}>确认</button>
            <button onClick={() => setSplitPoint(null)}>取消</button>
          </div>
        )}
      </div>

      {chunk.merged_from && (
        <div className="chunk-badge merged">
          已合并 {chunk.merged_from.length} 个片段
        </div>
      )}
      
      {chunk.split_from && (
        <div className="chunk-badge split">
          从片段 {chunk.split_from} 拆分
        </div>
      )}
```

## 6. 向量化与知识库一体化架构优化

### 6.1 当前割裂问题分析

通过对现有代码的深入分析，发现向量化管理模块与知识库其他模块存在以下割裂问题：

#### 6.1.1 数据层面割裂

| 问题 | 现状 | 影响 |
|------|------|------|
| 向量与实体分离 | 向量存储在 ChromaDB，实体存储在 PostgreSQL | 无法通过实体快速定位相关向量 |
| 片段与图谱分离 | 文本片段和知识图谱独立存储 | 无法利用图谱关系增强检索 |
| 版本管理不一致 | 文档版本与向量版本不同步 | 更新文档后向量可能过期 |
| 元数据分散 | 各模块维护自己的元数据 | 数据冗余，一致性难保证 |

#### 6.1.2 流程层面割裂

```
当前处理流程（割裂）：
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  文档解析   │───▶│  向量化     │    │  实体提取   │    │  图谱构建   │
│  (Parser)   │    │  (Chroma)   │    │  (Spacy)    │    │  (NetworkX) │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
       │                  │                  │                  │
       ▼                  ▼                  ▼                  ▼
  PostgreSQL         ChromaDB          PostgreSQL         PostgreSQL
  (文档表)            (向量表)           (实体表)           (关系表)
```

**问题**：
1. 各步骤独立执行，缺乏统一事务管理
2. 失败时难以回滚，容易产生脏数据
3. 无法利用前一步的结果优化后一步
4. 重复处理相同内容，浪费资源

#### 6.2.2 统一处理流水线

```python
# services/unified_processing_pipeline.py

from typing import Dict, Any, List, Optional
from sqlalchemy.orm import Session
import logging
from datetime import datetime
from enum import Enum

logger = logging.getLogger(__name__)


class ProcessingStage(Enum):
    """处理阶段枚举"""
    PARSE = "parse"
    CHUNK = "chunk"
    VECTORIZE = "vectorize"
    EXTRACT_ENTITIES = "extract_entities"
    BUILD_GRAPH = "build_graph"
    CREATE_ASSOCIATIONS = "create_associations"
    QUALITY_CHECK = "quality_check"


class UnifiedProcessingPipeline:
    """
    统一处理流水线
    
    将文档解析、分块、向量化、实体提取、图谱构建整合为
    一个连贯的处理流程，实现数据共享和事务管理
    """
    
    def __init__(self, db: Session):
        self.db = db
        self.stage_handlers = {
            ProcessingStage.PARSE: self._handle_parse,
            ProcessingStage.CHUNK: self._handle_chunk,
            ProcessingStage.VECTORIZE: self._handle_vectorize,
            ProcessingStage.EXTRACT_ENTITIES: self._handle_extract_entities,
            ProcessingStage.BUILD_GRAPH: self._handle_build_graph,
            ProcessingStage.CREATE_ASSOCIATIONS: self._handle_create_associations,
            ProcessingStage.QUALITY_CHECK: self._handle_quality_check,
        }
        
        # 初始化各服务
        from app.services.knowledge.document_parser import DocumentParser
        from app.services.knowledge.advanced_text_processor import AdvancedTextProcessor
        from app.services.knowledge.chroma_service import ChromaService
        from app.services.knowledge.knowledge_graph_service import KnowledgeGraphService
        
        self.parser = DocumentParser()
        self.text_processor = AdvancedTextProcessor(db)
        self.chroma_service = ChromaService()
        self.kg_service = KnowledgeGraphService()
    
    async def process_document(
        self,
        document_id: int,
        knowledge_base_id: int,
        file_path: str,
        file_type: str,
        config: Optional[Dict[str, Any]] = None
    ) -> Dict[str, Any]:
        """
        执行完整的文档处理流水线
        
        Args:
            document_id: 文档ID
            knowledge_base_id: 知识库ID
            file_path: 文件路径
            file_type: 文件类型
            config: 处理配置
            
        Returns:
            处理结果，包含各阶段产出和统计信息
        """
        config = config or {}
        pipeline_run = None
        
        try:
            # 1. 创建流水线运行记录
            pipeline_run = ProcessingPipelineRun(
                document_id=document_id,
                knowledge_base_id=knowledge_base_id,
                pipeline_config=config,
                status='running',
                started_at=datetime.now(),
                total_stages=len(self.stage_handlers)
            )
            self.db.add(pipeline_run)
            self.db.flush()
            
            # 2. 按顺序执行各阶段
            context = {
                'document_id': document_id,
                'knowledge_base_id': knowledge_base_id,
                'file_path': file_path,
                'file_type': file_type,
                'config': config,
                'pipeline_run_id': pipeline_run.id,
                'intermediate_results': {}
            }
            
            for stage in ProcessingStage:
                pipeline_run.current_stage = stage.value
                self.db.commit()
                
                logger.info(f"[Pipeline] 开始执行阶段: {stage.value}")
                
                try:
                    result = await self.stage_handlers[stage](context)
                    context['intermediate_results'][stage.value] = result
                    
                    # 更新阶段结果
                    setattr(pipeline_run, f"{stage.value}_result", {
                        'success': True,
                        'data': result,
                        'timestamp': datetime.now().isoformat()
                    })
                    
                    pipeline_run.completed_stages += 1
                    self.db.commit()
                    
                    logger.info(f"[Pipeline] 阶段 {stage.value} 完成")
                    
                except Exception as e:
                    logger.error(f"[Pipeline] 阶段 {stage.value} 失败: {str(e)}")
                    
                    pipeline_run.status = 'failed'
                    pipeline_run.error_stage = stage.value
                    pipeline_run.error_message = str(e)
                    self.db.commit()
                    
                    # 根据配置决定是否回滚
                    if config.get('rollback_on_failure', True):
                        await self._rollback_document_processing(document_id)
                    
                    raise
            
            # 3. 流水线完成
            pipeline_run.status = 'completed'
            pipeline_run.completed_at = datetime.now()
            pipeline_run.total_duration_ms = (
                pipeline_run.completed_at - pipeline_run.started_at
            ).total_seconds() * 1000
            self.db.commit()
            
            logger.info(f"[Pipeline] 文档 {document_id} 处理完成")
            
            return {
                'success': True,
                'pipeline_run_id': pipeline_run.id,
                'results': context['intermediate_results']
            }
            
        except Exception as e:
            logger.error(f"[Pipeline] 处理失败: {str(e)}")
            if pipeline_run:
                pipeline_run.status = 'failed'
                pipeline_run.error_message = str(e)
                self.db.commit()
            raise
    
    async def _handle_parse(self, context: Dict[str, Any]) -> Dict[str, Any]:
        """文档解析阶段"""
        file_path = context['file_path']
        file_type = context['file_type']
        
        # 解析文档
        raw_text = self.parser.parse_document(file_path)
        
        # 提取文档元数据
        metadata = self.parser.extract_metadata(file_path, file_type)
        
        return {
            'raw_text': raw_text,
            'metadata': metadata,
            'text_length': len(raw_text)
        }
    
    async def _handle_chunk(self, context: Dict[str, Any]) -> Dict[str, Any]:
        """智能分块阶段"""
        parse_result = context['intermediate_results']['parse']
        raw_text = parse_result['raw_text']
        config = context['config']
        
        # 文本清理
        cleaned_text = self.text_processor.clean_text(raw_text)
        
        # 语义分块
        chunks = self.text_processor.semantic_chunking_sync(
            cleaned_text,
            chunk_size=config.get('chunk_size', 500),
            chunk_overlap=config.get('chunk_overlap', 50)
        )
        
        # 为每个分块创建知识单元（暂不提交）
        knowledge_units = []
        for idx, chunk_text in enumerate(chunks):
            unit = UnifiedKnowledgeUnit(
                unit_type='chunk',
                knowledge_base_id=context['knowledge_base_id'],
                document_id=context['document_id'],
                content=chunk_text,
                chunk_index=idx,
                total_chunks=len(chunks),
                context_window=self._extract_context_window(cleaned_text, chunk_text, idx),
                processing_stage='chunked'
            )
            knowledge_units.append(unit)
        
        # 暂存到上下文，后续阶段使用
        context['knowledge_units'] = knowledge_units
        
        return {
            'chunks': chunks,
            'chunk_count': len(chunks),
            'knowledge_units': [unit.id for unit in knowledge_units]
        }
    
    async def _handle_vectorize(self, context: Dict[str, Any]) -> Dict[str, Any]:
        """向量化阶段"""
        knowledge_units = context.get('knowledge_units', [])
        config = context['config']
        
        if not knowledge_units:
            return {'vectorized_count': 0}
        
        # 准备批量向量化数据
        documents_for_chroma = []
        for unit in knowledge_units:
            documents_for_chroma.append({
                'document_id': f"chunk_{unit.document_id}_{unit.chunk_index}",
                'text': unit.content,
                'metadata': {
                    'knowledge_base_id': unit.knowledge_base_id,
                    'document_id': unit.document_id,
                    'chunk_index': unit.chunk_index,
                    'unit_id': unit.id
                }
            })
        
        # 批量向量化
        vector_result = self.chroma_service.add_documents_batch(documents_for_chroma)
        
        if vector_result.get('success'):
            # 更新知识单元的向量信息
            for i, unit in enumerate(knowledge_units):
                unit.content_vector_id = documents_for_chroma[i]['document_id']
                unit.embedding_model = config.get('embedding_model', 'default')
                unit.embedding_version = 1
                unit.processing_stage = 'vectorized'
                self.db.add(unit)
            
            self.db.commit()
        
        return {
            'vectorized_count': vector_result.get('count', 0),
            'vector_ids': [d['document_id'] for d in documents_for_chroma]
        }
    
    async def _handle_extract_entities(self, context: Dict[str, Any]) -> Dict[str, Any]:
        """实体提取阶段 - 利用向量化的中间结果"""

    async def _handle_create_associations(self, context: Dict[str, Any]) -> Dict[str, Any]:
        """创建知识单元关联阶段"""
        knowledge_base_id = context['knowledge_base_id']
        document_id = context['document_id']
        
        # 获取该文档的所有知识单元
        units = self.db.query(UnifiedKnowledgeUnit).filter(
            UnifiedKnowledgeUnit.document_id == document_id
        ).all()
        
        associations_created = 0
        
        # 1. 创建实体-片段关联（实体出现在哪些片段中）
        chunk_units = [u for u in units if u.unit_type == 'chunk']
        entity_units = [u for u in units if u.unit_type == 'entity']
        
        for entity_unit in entity_units:
            for chunk_unit in chunk_units:
                if entity_unit.content in chunk_unit.content:
                    association = KnowledgeUnitAssociation(
                        source_unit_id=entity_unit.id,
                        target_unit_id=chunk_unit.id,
                        association_type='entity_in_chunk',
                        association_strength=1.0,
                        association_details={
                            'entity_type': entity_unit.entity_type,
                            'chunk_index': chunk_unit.chunk_index
                        }
                    )
                    self.db.add(association)
                    associations_created += 1
        
        # 2. 创建向量相似度关联
        # 查询相似向量并创建关联
        for chunk_unit in chunk_units:
            if chunk_unit.content_vector_id:
                similar_results = self.chroma_service.similarity_search(
                    chunk_unit.content,
                    n_results=5,
                    filter={
                        'knowledge_base_id': knowledge_base_id,
                        'document_id': {'$ne': document_id}  # 排除当前文档
                    }
                )
                
                for result in similar_results:
                    if result['distance'] < 0.3:  # 相似度阈值
                        # 查找对应的知识单元
                        target_unit = self.db.query(UnifiedKnowledgeUnit).filter(
                            UnifiedKnowledgeUnit.content_vector_id == result['id']
                        ).first()
                        
                        if target_unit:
                            association = KnowledgeUnitAssociation(
                                source_unit_id=chunk_unit.id,
                                target_unit_id=target_unit.id,
                                association_type='vector_similarity',
                                association_strength=1 - result['distance'],
                                association_details={
                                    'distance': result['distance'],
                                    'similarity_score': 1 - result['distance']
                                }
                            )
                            self.db.add(association)
                            associations_created += 1
        
        self.db.commit()
        
        return {
            'associations_created': associations_created
        }
    
    async def _handle_quality_check(self, context: Dict[str, Any]) -> Dict[str, Any]:
        """质量检查阶段"""
        document_id = context['document_id']
        
        # 获取所有知识单元
        units = self.db.query(UnifiedKnowledgeUnit).filter(
            UnifiedKnowledgeUnit.document_id == document_id
        ).all()
        
        quality_results = []
        
        for unit in units:
            if unit.unit_type == 'chunk':
                # 计算片段质量
                coherence = self._calculate_coherence(unit)
                distinctiveness = self._calculate_distinctiveness(unit, units)
                
                unit.coherence_score = coherence
                unit.distinctiveness_score = distinctiveness
                unit.quality_score = (coherence + distinctiveness) / 2
                unit.processing_stage = 'quality_checked'
                
                quality_results.append({
                    'unit_id': unit.id,
                    'quality_score': unit.quality_score,
                    'coherence': coherence,
                    'distinctiveness': distinctiveness
                })
        
        self.db.commit()
        
        return {
            'units_checked': len(quality_results),
            'average_quality': sum(r['quality_score'] for r in quality_results) / len(quality_results) if quality_results else 0,
            'details': quality_results
        }
    
    def _extract_context_window(self, full_text: str, chunk_text: str, chunk_index: int, 
                                 window_size: int = 100) -> str:
        """提取片段的上下文窗口"""
        chunk_start = full_text.find(chunk_text)
        if chunk_start == -1:
            return ""
        
        context_start = max(0, chunk_start - window_size)
        context_end = min(len(full_text), chunk_start + len(chunk_text) + window_size)
        
        return full_text[context_start:context_end]
    
    def _calculate_coherence(self, unit: UnifiedKnowledgeUnit) -> float:
        """计算片段连贯性分数"""
        # 简化的连贯性计算：检查片段长度和句子完整性
        content = unit.content
        
        if not content:
            return 0.0
        
        # 基于长度的评分
        length_score = min(1.0, len(content) / 500)  # 理想长度约500字符
        
        # 句子完整性评分
        sentence_count = content.count('。') + content.count('.') + content.count('!') + content.count('?')
        completeness_score = min(1.0, sentence_count / 3)  # 理想包含3个句子
        
        return (length_score + completeness_score) / 2
    
    def _calculate_distinctiveness(self, unit: UnifiedKnowledgeUnit, 
                                    all_units: List[UnifiedKnowledgeUnit]) -> float:
        """计算片段区分度分数"""
        # 简化的区分度计算：与同一文档中其他片段的差异
        content = unit.content
        
        if not content or len(all_units) <= 1:
            return 1.0
        
        # 计算与其他片段的文本相似度（使用简单的Jaccard系数）
        similarities = []
        unit_words = set(content.split())
        
        for other in all_units:
            if other.id != unit.id and other.unit_type == 'chunk':
                other_words = set(other.content.split())
                if unit_words and other_words:
                    intersection = len(unit_words & other_words)
                    union = len(unit_words | other_words)
                    similarity = intersection / union if union > 0 else 0
                    similarities.append(similarity)
        
        # 区分度 = 1 - 平均相似度
        avg_similarity = sum(similarities) / len(similarities) if similarities else 0
        return 1 - avg_similarity
```

```python
# services/unified_retrieval_service.py

from typing import Dict, Any, List, Optional, Tuple
from sqlalchemy.orm import Session
from sqlalchemy import func, or_, and_
import logging

logger = logging.getLogger(__name__)


class UnifiedRetrievalService:
    """
    一体化检索服务
    
    整合向量检索、实体检索、图谱检索，提供统一的检索接口
    """
    
    def __init__(self, db: Session):
        self.db = db
        
        from app.services.knowledge.chroma_service import ChromaService
        from app.services.knowledge.knowledge_graph_service import KnowledgeGraphService
        from app.services.knowledge.semantic_search_service import SemanticSearchService

    async def unified_search(
        self,
        query: str,
        knowledge_base_id: Optional[int] = None,
        search_modes: List[str] = None,
        n_results: int = 10,
        filters: Optional[Dict[str, Any]] = None
    ) -> Dict[str, Any]:
        """
        统一检索接口
        
        Args:
            query: 查询字符串
            knowledge_base_id: 知识库ID（可选）
            search_modes: 检索模式列表 ['vector', 'entity', 'graph', 'semantic']
            n_results: 返回结果数量
            filters: 过滤条件
            
        Returns:
            整合后的检索结果
        """
        search_modes = search_modes or ['vector', 'entity', 'semantic']
        filters = filters or {}
        
        results = {
            'query': query,
            'search_modes': search_modes,
            'results': [],
            'sources': {}
        }
        
        # 1. 向量检索
        if 'vector' in search_modes:
            vector_results = await self._vector_search(
                query, knowledge_base_id, n_results * 2, filters
            )
            results['sources']['vector'] = vector_results
        
        # 2. 实体检索
        if 'entity' in search_modes:
            entity_results = await self._entity_search(
                query, knowledge_base_id, n_results * 2
            )
            results['sources']['entity'] = entity_results
        
        # 3. 图谱检索
        if 'graph' in search_modes:
            graph_results = await self._graph_search(
                query, knowledge_base_id, n_results
            )
            results['sources']['graph'] = graph_results
        
        # 4. 语义检索
        if 'semantic' in search_modes:
            semantic_results = self.semantic_service.semantic_search(
                query=query,
                n_results=n_results * 2,
                knowledge_base_id=knowledge_base_id
            )
            results['sources']['semantic'] = semantic_results
        
        # 5. 融合排序
        fused_results = self._fuse_results(
            results['sources'],
            n_results,
            fusion_strategy='rrf'  # Reciprocal Rank Fusion
        )
        
        results['results'] = fused_results
        results['total_count'] = len(fused_results)
        
        return results
    
    async def _vector_search(

    def _fuse_results(
        self,
        sources: Dict[str, List[Dict[str, Any]]],
        n_results: int,
        fusion_strategy: str = 'rrf'
    ) -> List[Dict[str, Any]]:
        """
        融合多源检索结果
        
        Args:
            sources: 各检索源的结果
            n_results: 返回结果数
            fusion_strategy: 融合策略 ('rrf', 'weighted', 'linear')
        """
        if fusion_strategy == 'rrf':
            return self._reciprocal_rank_fusion(sources, n_results)
        elif fusion_strategy == 'weighted':
            return self._weighted_fusion(sources, n_results)
        else:
            return self._linear_fusion(sources, n_results)
    
    def _reciprocal_rank_fusion(
        self,
        sources: Dict[str, List[Dict[str, Any]]],
        n_results: int,
        k: int = 60
    ) -> List[Dict[str, Any]]:
        """
        倒数排名融合 (Reciprocal Rank Fusion)
        
        RRF 分数 = Σ(1 / (k + rank))
        """
        from collections import defaultdict
        
        # 收集所有文档的分数
        doc_scores = defaultdict(float)
        doc_info = {}
        
        for source_name, results in sources.items():
            for rank, result in enumerate(results, start=1):
                doc_id = result.get('id')
                if doc_id:
                    # RRF 公式
                    doc_scores[doc_id] += 1 / (k + rank)
                    
                    # 保存文档信息（使用第一次出现的）
                    if doc_id not in doc_info:
                        doc_info[doc_id] = result
        
        # 按分数排序
        sorted_docs = sorted(
            doc_scores.items(),
            key=lambda x: x[1],
            reverse=True
        )
        
        # 构建最终结果
        fused_results = []
        for doc_id, score in sorted_docs[:n_results]:
            result = doc_info[doc_id].copy()
            result['fused_score'] = score
            result['fusion_method'] = 'rrf'
            fused_results.append(result)
        
        return fused_results
    
    def _weighted_fusion(
        self,
        sources: Dict[str, List[Dict[str, Any]]],
        n_results: int
    ) -> List[Dict[str, Any]]:
        """加权融合"""
        # 定义各源的权重
        weights = {
            'vector': 0.4,
            'entity': 0.3,
            'semantic': 0.2,
            'graph': 0.1
        }
        
        from collections import defaultdict
        
        doc_scores = defaultdict(float)
        doc_info = {}
        
        for source_name, results in sources.items():
            weight = weights.get(source_name, 0.1)
            
            for result in results:
                doc_id = result.get('id')
                if doc_id:
                    # 加权分数
                    original_score = result.get('score', 0)
                    doc_scores[doc_id] += original_score * weight
                    
                    if doc_id not in doc_info:
                        doc_info[doc_id] = result
        
        # 排序并返回
        sorted_docs = sorted(
            doc_scores.items(),
            key=lambda x: x[1],
            reverse=True
        )
        
        fused_results = []
        for doc_id, score in sorted_docs[:n_results]:
            result = doc_info[doc_id].copy()
            result['fused_score'] = score
            result['fusion_method'] = 'weighted'
            fused_results.append(result)
        
        return fused_results
    
    async def get_document_knowledge_graph(
        self,

        include_vectors: bool = True,

    ) -> Dict[str, Any]:
        """
        获取文档的完整知识图谱视图
        
        整合向量、实体、关系、关联，提供文档的全局视图
        """
        result = {
            'document_id': document_id,
            'nodes': [],
            'edges': [],
            'statistics': {}
        }
        
        # 1. 获取知识单元
        units = self.db.query(UnifiedKnowledgeUnit).filter(
            UnifiedKnowledgeUnit.document_id == document_id
        ).all()
        
        # 2. 构建节点
        for unit in units:
            node = {
                'id': unit.id,
                'type': unit.unit_type,
                'content': unit.content[:200] if unit.content else '',
                'entity_type': unit.entity_type,
                'quality_score': unit.quality_score
            }
            
            if include_vectors and unit.content_vector_id:
                node['has_vector'] = True
                node['vector_id'] = unit.content_vector_id
            
            result['nodes'].append(node)
        
        # 3. 获取关联
        if include_associations:
            unit_ids = [u.id for u in units]
            
            associations = self.db.query(KnowledgeUnitAssociation).filter(
                KnowledgeUnitAssociation.source_unit_id.in_(unit_ids)
            ).all()
            
            for assoc in associations:
                edge = {
                    'source': assoc.source_unit_id,
                    'target': assoc.target_unit_id,
                    'type': assoc.association_type,
                    'strength': assoc.association_strength
                }
                result['edges'].append(edge)
        
        # 4. 统计信息
        result['statistics'] = {
            'total_units': len(units),
            'chunk_count': len([u for u in units if u.unit_type == 'chunk']),
            'entity_count': len([u for u in units if u.unit_type == 'entity']),
            'relation_count': len([u for u in units if u.unit_type == 'relation']),
            'association_count': len(result['edges']),
            'vectorized_count': len([u for u in units if u.content_vector_id])
        }
        
        return result
```

|------|------|----------|

### 5.2 实施建议

1. **渐进式改造**：避免一次性大规模重构，采用增量优化策略
2. **充分测试**：每个优化点都需要配套的性能测试和回归测试
3. **监控先行**：在实施优化前先建立完善的监控体系
4. **文档同步**：及时更新技术文档，确保团队知识同步
