## 三、数据模型支持

### 3.1 三层实体模型（已实现）

```python
# DocumentEntity - 文档级实体
class DocumentEntity(Base):
    id = Column(Integer, primary_key=True)
    entity_text = Column(String(200))      # 实体文本
    entity_type = Column(String(50))       # 实体类型
    kb_entity_id = Column(ForeignKey('kb_entities.id'))  # 关联KB实体

# KBEntity - 知识库级实体
class KBEntity(Base):
    id = Column(Integer, primary_key=True)
    knowledge_base_id = Column(Integer)
    canonical_name = Column(String(200))   # 规范名称
    entity_type = Column(String(50))
    aliases = Column(JSON)                 # 别名列表
    global_entity_id = Column(ForeignKey('global_entities.id'))

# GlobalEntity - 全局级实体
class GlobalEntity(Base):
    id = Column(Integer, primary_key=True)
    global_name = Column(String(200))      # 全局规范名称
    entity_type = Column(String(50))
    all_aliases = Column(JSON)             # 所有别名
```
