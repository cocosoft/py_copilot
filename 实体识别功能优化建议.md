### 3.2 功能增强（中优先级）

#### 问题3：缺少批量操作功能

**现状**：实体只能逐个确认/拒绝/删除，效率较低。

**优化建议**：添加批量操作功能

```jsx
// 新增状态
const [selectedEntities, setSelectedEntities] = useState([]);

// 批量操作函数
const handleSelectAllEntities = () => {
  setSelectedEntities(filteredEntities.map(e => e.id));
};

const handleClearEntitySelection = () => {
  setSelectedEntities([]);
};

const handleBatchConfirm = () => {
  setEntities(prev =>
    prev.map(e =>
      selectedEntities.includes(e.id) && e.status === 'pending'
        ? { ...e, status: 'confirmed' }
        : e
    )
  );
  setSelectedEntities([]);
  message.success({ content: `已批量确认 ${selectedEntities.length} 个实体` });
};

const handleBatchReject = () => {
  setEntities(prev =>
    prev.map(e =>
      selectedEntities.includes(e.id) && e.status === 'pending'
        ? { ...e, status: 'rejected' }
        : e
    )
  );
  setSelectedEntities([]);
  message.success({ content: `已批量拒绝 ${selectedEntities.length} 个实体` });
};
```

**UI 组件**：

```jsx
{/* 批量操作栏 */}
```
