# Py Copilot 工具功能设计方案

## 概述

本文档详细设计了 Py Copilot 应用中所有工具的功能规格，包括现有工具的完善方案和新工具的设计建议。

---

## 一、现有工具功能完善方案

### 1. Web搜索工具 (WebSearchTool)

#### 当前状态
- ✅ 基础搜索功能已实现
- ⚠️ 需要完善错误处理和结果格式化

#### 功能设计

**核心功能：**
1. **多引擎搜索支持**
   - Google Search API
   - Bing Search API
   - Baidu Search API
   - DuckDuckGo (无需API Key)

2. **搜索参数：**
   - `query` (string, required): 搜索查询词
   - `engine` (string, optional): 搜索引擎，默认 "google"
   - `num_results` (integer, optional): 返回结果数量 (1-20)
   - `safe_search` (boolean, optional): 安全搜索模式，默认 true
   - `time_range` (string, optional): 时间范围 (day, week, month, year)
   - `site` (string, optional): 限定搜索站点

3. **返回结果格式：**
   ```json
   {
     "success": true,
     "results": [
       {
         "title": "结果标题",
         "url": "https://example.com",
         "snippet": "结果摘要...",
         "source": "google",
         "timestamp": "2024-01-15"
       }
     ],
     "total_results": 100,
     "search_time": 0.523
   }
   ```

4. **增强功能：**
   - 搜索结果缓存（TTL: 1小时）
   - 搜索结果去重
   - 结果相关性排序
   - 搜索历史记录

#### 待实现功能
- [ ] 搜索结果摘要生成
- [ ] 搜索结果导出（JSON/CSV）
- [ ] 批量搜索支持
- [ ] 搜索结果可视化

### 2. 知识库搜索工具 (KnowledgeSearchTool)

#### 当前状态
- ✅ 基础搜索功能已实现
- ⚠️ 需要优化向量搜索和结果相关性

#### 功能设计

**核心功能：**
1. **多知识库支持**

2. **搜索参数：**
   - `query` (string, required): 搜索查询词

   - `top_k` (integer, optional): 返回结果数量，默认 5

     - `date_range`: 日期范围过滤
     - `tags`: 标签过滤

3. **返回结果格式：**
   ```json
   {
     "success": true,
     "results": [
       {
         "content": "匹配的内容片段...",
         "document_id": "doc_123",
         "document_title": "文档标题",
         "knowledge_base_name": "知识库名称",
         "similarity_score": 0.89,
         "metadata": {
           "page": 5,
           "chunk_index": 3
         }
       }
     ],
     "total_found": 25,
     "search_time": 0.234
   }
   ```

4. **增强功能：**
   - 语义搜索（向量相似度）
   - 关键词搜索（BM25）
   - 混合搜索（语义+关键词）
   - 搜索结果高亮

#### 待实现功能

## 三、工具管理系统设计

### 1. 工具注册与发现

**工具注册流程：**
1. 工具类继承 `BaseTool`
2. 实现 `_get_metadata()` 方法
3. 实现 `_get_parameters()` 方法
4. 实现 `execute()` 方法
