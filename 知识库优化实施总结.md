```python
while retry_count < max_retries:
    try:
        # 处理逻辑...
        break  # 成功则跳出循环
    except Exception as e:
        retry_count += 1
        if retry_count >= max_retries:
            raise
        wait_time = 2 ** retry_count
        await asyncio.sleep(wait_time)
```

## 模型依赖

### 需要下载的模型
1. **BAAI/bge-large-zh-v1.5** - Embedding模型
2. **BAAI/bge-reranker-large** - 重排序模型
3. **zh_core_web_sm** - spaCy中文NER模型

### 模型下载命令
```bash
# Embedding和重排序模型（自动下载）
# 首次使用时会自动从HuggingFace下载

# spaCy中文模型
python -m spacy download zh_core_web_sm
```

### 模型大小
- BAAI/bge-large-zh-v1.5: ~1.3GB
- BAAI/bge-reranker-large: ~1.2GB
- zh_core_web_sm: ~50MB
- **总计**: ~2.55GB
