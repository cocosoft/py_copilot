### 2. Agent系统集成模块

**文件位置**: `backend/app/agents/integration.py`

**核心功能**:
- 与现有数据库Agent模型集成
- 从数据库同步Agent到注册中心
- 统一执行接口
- 执行日志记录
- 任务匹配和推荐

**主要方法**:
- `initialize()`: 初始化集成
- `execute_agent()`: 执行Agent
- `get_agent_info()`: 获取Agent信息
- `list_agents()`: 列出Agent
- `find_agents_for_task()`: 为任务查找Agent

### 3. Agent编排服务

**文件位置**: `backend/app/services/agent_orchestration_service.py`

**核心功能**:
- 整合Agent系统、能力中心和编排引擎
- 智能请求路由（Agent执行、能力执行、复杂编排）
- 流式处理支持
- 系统状态监控

**处理流程**:
1. 理解用户意图
2. 根据意图类型选择处理方式:
   - `agent_execution`: 直接执行Agent
   - `capability_execution`: 执行能力
   - 其他: 使用编排引擎处理复杂任务

**主要方法**:
- `process_request()`: 处理用户请求
- `stream_process_request()`: 流式处理
- `execute_agent_directly()`: 直接执行Agent
- `get_system_status()`: 获取系统状态

### 4. Agent API路由

**文件位置**: `backend/app/api/v1/endpoints/agents.py`

**提供的接口**:

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/agents/list` | 获取Agent列表 |
| GET | `/agents/detail/{agent_name}` | 获取Agent详情 |
| POST | `/agents/execute/{agent_name}` | 执行指定Agent |
| POST | `/agents/chat` | 智能对话（自动路由） |
| GET | `/agents/search` | 搜索Agent |
| GET | `/agents/status` | 获取系统状态 |
| POST | `/agents/find-for-task` | 为任务查找Agent |
| GET | `/agents/official` | 获取官方Agent列表 |
| GET | `/agents/categories` | 获取Agent分类 |
| WebSocket | `/agents/ws/chat` | WebSocket对话 |

## 使用示例

### 初始化Agent系统
```bash
python scripts/init_agent_system.py
```

### 调用Agent API
```python
import requests

# 列出Agent
response = requests.get("/api/v1/agents/list")
agents = response.json()

# 执行Agent
response = requests.post("/api/v1/agents/execute/聊天助手", json={
    "input": "你好",
    "session_id": "session_123"
})
result = response.json()

# 智能对话
response = requests.post("/api/v1/agents/chat", json={
    "message": "翻译这段文字",
    "context": {}
})
result = response.json()
```

### 使用编排服务
```python
from app.services.agent_orchestration_service import get_orchestration_service

service = get_orchestration_service(db)

# 处理请求
result = await service.process_request(
    user_input="生成一张猫的图片",
    user_id="user_123",
    session_id="session_456"
)

# 流式处理
async for chunk in service.stream_process_request("你好"):
    print(chunk)
```
