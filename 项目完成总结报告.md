## 四、技术栈

### 后端
- **框架**: FastAPI
- **数据库**: SQLite/PostgreSQL + SQLAlchemy
- **缓存**: Redis
- **AI/ML**: spaCy, LangChain, scikit-learn
- **向量存储**: ChromaDB/FAISS
- **测试**: pytest

### 前端
- **框架**: React 18
- **构建工具**: Vite
- **状态管理**: Zustand
- **可视化**: D3.js, ReactFlow
- **UI组件**: 自定义统一组件库
- **测试**: Jest, React Testing Library
